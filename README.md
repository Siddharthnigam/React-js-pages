## Installation and Setup

### Clone the Repository
```bash
git clone <repository-url>
cd Interview/project
```

### Install Dependencies
```bash
npm install
```

### Run the Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
